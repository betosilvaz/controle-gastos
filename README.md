# Sistem de Gestão de Gastos

O projeto se trata de um sistema de controle de gastos, onde você pode cadastrar pessoas e transações relacionadas a elas. O sistema também possui dashboard que mostra tudo que entrou, saiu e a balança total de cada pessoa e de todo o sistema.

## Tecnologias

**Backend:** C# & Dotnet<br>
**Frontend:** Typescript, React, Tanstack Router, React Hot Toaster

## Como rodar

**Backend**
```
dotnet restore
dotnet run
```

**Frontend**
```
npm install
npm run dev
```
